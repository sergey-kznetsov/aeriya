# Packs

Эта папка предназначена для будущих compendium packs модуля Аэрии.

Старые compendium packs из `shadow-lands` сюда не копируются. Модуль пересобирается с нуля и должен содержать только материалы, связанные с миром Аэрии.

## Правила для паков

В packs добавляются только очищенные и проверенные материалы:

```text
города Аэрии;
поселения и стоянки Аэрии;
фракции Аэрии;
духи Аэрии;
НИПы Аэрии;
монстры и бестиарии Аэрии;
ритуалы Аэрии;
предметы и артефакты Аэрии;
книги, байки, предания, легенды и пророчества Аэрии;
таблицы событий, слухов, примет и откатов Аэрии.
```

Не добавляются:

```text
старые внешние компендиумы;
материалы, не связанные с текущим лором;
данные готового мира;
пользователи;
история чата;
fog of war;
аудио;
карты;
токены;
портреты;
служебные файлы Foundry и LevelDB без проверки.
```

## Планируемые пакеты

```text
aeriya-journals
aeriya-cities
aeriya-factions
aeriya-spirits
aeriya-rituals
aeriya-npcs
aeriya-monsters
aeriya-items
aeriya-scenes
aeriya-rolltables
```

`module.json` должен объявлять пакеты только после того, как они реально подготовлены и проверены в чистом мире Foundry.
