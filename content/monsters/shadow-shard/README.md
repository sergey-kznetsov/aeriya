# Монстры Теневого Осколка

Здесь будут лежать существа Пепельной Степи, Леса Чёрного Шёпота и Ледяного / Морозного Предела.

Внутреннее деление:

```text
ash-steppe/              — пепельные, степные и буревые угрозы
black-whisper-forest/    — лесные твари, культы корней, грибные и теневые существа
icebound-reaches/        — ледяные существа, духи холода, угрозы Йормунги
```

Каждый монстр оформляется по шаблону:

```text
content/_templates/monster.md
```
