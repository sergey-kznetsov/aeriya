# Монстры Срединных Земель

Здесь будут лежать существа, угрозы, бандиты, аномалии, болотные твари, подземные существа и городские противники Срединных Земель.

Каждый монстр оформляется по шаблону:

```text
content/_templates/monster.md
```

В карточке обязательно указывать слабые места, точные попадания, тактику и проверки, через которые игроки могут узнать анатомию или уязвимость существа.
