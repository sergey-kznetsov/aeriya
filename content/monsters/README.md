# Монстры и бестиарии

Раздел для монстров, существ, боссов, духовных угроз, нежити, конструктов, тварей Извне и региональных опасностей.

Монстры должны быть разделены по региону и типу угрозы. Для боя особенно важно фиксировать слабые места и возможные точные попадания.

Рекомендуемые подпапки:

```text
by-region/
by-type/
bosses/
```

Рекомендуемые типы:

```text
beasts/
undead/
spirits/
constructs/
aberrations/
humanoids/
regional-threats/
```

Карточка монстра должна содержать регион, тип, роль в бою, поведение, слабые места, особые атаки, реакции, возможные точные попадания, трофеи, слухи и связанные локации.
