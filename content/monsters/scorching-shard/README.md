# Монстры Палящего Осколка

Здесь будут лежать существа Выжженных пустошей, Соляных пустынь, Огненных каньонов и подземных зон Палящего Осколка.

Внутреннее деление:

```text
burnt-wastes/     — пустынные твари, солнечные угрозы, мутанты
salt-deserts/     — соляные существа, миражи, стражи, провалы
fire-canyons/     — огненные существа, лавовые и стеклянные угрозы
underground/      — подземные существа, хранители, слепые угрозы
```

Каждый монстр оформляется по шаблону:

```text
content/_templates/monster.md
```
