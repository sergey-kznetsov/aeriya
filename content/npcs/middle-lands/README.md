# НИПы Срединных Земель

Здесь будут лежать карточки НИПов Срединных Земель.

НИПы группируются по основному месту появления или основной фракции. Если персонаж связан с несколькими городами, основной файл хранится здесь, а ссылки добавляются в карточки городов и фракций.
