# НИПы

Раздел для важных неигровых персонажей Аэрии: лидеров, торговцев, проводников, врагов, покровителей, свидетелей, должников, жрецов, шаманов и случайных встреч.

НИПы должны быть связаны с городом, фракцией, регионом и сценами. Это нужно, чтобы мастер мог быстро понять, где персонаж появляется и зачем он нужен в игре.

Рекомендуемые подпапки:

```text
by-city/
by-faction/
merchants/
guides/
enemies/
patrons/
```

Карточка NPC должна содержать имя, роль, город или регион, связанную фракцию, мотивацию, страх, тайну, пользу для партии, манеру речи, боевой статус и связанные квесты.
