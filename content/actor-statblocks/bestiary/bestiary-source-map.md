# Карта переноса чарников бестиария

## Назначение

Этот файл фиксирует, какие чарники бестиария нужно переносить из исходных книг, а какие существа были добавлены в модуль позже и требуют нового actor-ready статблока.

## Главный принцип

Существующие бестиарные статблоки не придумываются заново. Они переносятся из исходных книг:

- `Книга_III_Бестиарий.docx` — Срединные Земли.
- `Книга_VII_Бестиарий_Теневого_и_Палящего_Осколков.docx` — Теневой и Палящий Осколки.
- региональные книги используются как дополнительные источники сверки.

Новые статблоки создаются только для существ, которые появились в модуле как новые угрозы и не имеют исходного чарника.

## Новые существа модуля, которым нужны собственные статблоки

Эти существа появились в новых encounter/city-scene слоях и не должны автоматически заменять исходных существ:

| Существо | Регион | Статус |
|---|---|---|
| Кровляный червь | Срединные Земли | needs-new-statblock |
| Болотный шатун | Срединные Земли | needs-new-statblock или замена на Болотную химеру по сцене |
| Колокольный ворон | Срединные Земли | needs-new-statblock или замена на Эхо-крыса/Слышащего сталкера по сцене |
| Складской хват | Срединные Земли | needs-new-statblock или замена на Пыльника-технопата по сцене |
| Сухоглазый сторож | Срединные Земли | needs-new-statblock или адаптация Тени |
| Химерный выкормок | Срединные Земли | needs-new-statblock или адаптация Модифицированного Пациента |
| Липовый роевик | Срединные Земли | needs-new-statblock |
| Семиглазая тень | Срединные Земли | needs-new-statblock или адаптация Тени |
| Грядовый прыгун | Срединные Земли | needs-new-statblock |

## Следующий порядок работы

1. Перенести source-ready чарники из `Книга_III_Бестиарий.docx`.
2. Перенести source-ready чарники из `Книга_VII_Бестиарий_Теневого_и_Палящего_Осколков.docx`.
3. Создать отдельные новые статблоки только для `needs-new-statblock`.
4. Добавить NPC statblocks партиями: сначала Срединные Земли, затем Теневой Осколок, затем Палящий Осколок.
