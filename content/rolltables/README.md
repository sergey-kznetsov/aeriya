# Таблицы

Раздел для случайных и полуавтоматических таблиц Аэрии: дорожные события, слухи, приметы, ритуальные откаты, фракционные реакции, встречи, осложнения и цены эпических действий.

Таблицы должны помогать мастеру быстро принимать решения во время игры, а не заменять мастерское решение.

Рекомендуемые подпапки:

```text
road-events/
rumors/
omens/
ritual-backlash/
faction-reactions/
monster-behavior/
epic-costs/
```

Карточка таблицы должна содержать:

```text
Название
Для чего используется
Регион
Когда бросать
Кто видит результат
Таблица результатов
Как интерпретировать
Связанные правила
Связанные handouts
```
