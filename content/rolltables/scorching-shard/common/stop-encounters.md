# Палящий Осколок — встречи на стоянке

## Назначение

Таблица помогает создать социальную сцену на стоянке, у навеса или у городских ворот.

## d8

| d8 | Событие |
|---:|---|
| 1 | Хозяин стоянки просит назвать цель пути. |
| 2 | Проводник ищет группу на короткий маршрут. |
| 3 | Мастер предлагает проверить дорожную линзу. |
| 4 | Караван ждёт решения старших. |
| 5 | Один из гостей оставил знак без объяснения. |
| 6 | Путник продаёт старую схему маршрута. |
| 7 | Спорят о времени выхода. |
| 8 | Снаружи подан короткий сигнал. |

RollTable-заготовка.
