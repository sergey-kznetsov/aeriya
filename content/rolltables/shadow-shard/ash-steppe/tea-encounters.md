# Пепельная Степь — встречи за чаем

## Назначение

Таблица помогает быстро создать социальную сцену вокруг степной чайной традиции. Число чаш показывает отношение к гостю и задаёт тон разговора.

## d10

| d10 | Событие |
|---:|---|
| 1 | Первая чаша подана строго по обычаю, без личного тепла. |
| 2 | Вторая чаша означает, что гостей признают, но ещё проверяют. |
| 3 | Третья чаша открывает разговор о возможном партнёрстве. |
| 4 | Четвёртая чаша означает личное доверие одного из хозяев. |
| 5 | Пятая чаша предлагается неожиданно и требует осторожного ответа. |
| 6 | Хозяин нарушает привычный порядок чаш, чтобы подать знак. |
| 7 | Один из гостей получает чашу отдельно от остальных. |
| 8 | Во время чаепития появляется человек с просьбой. |
| 9 | Хозяева задают вопрос, на который нельзя отвечать слишком прямо. |
| 10 | Чаепитие прерывается вестью с дороги. |

## Использование в Foundry

Карточка используется как черновик RollTable. Позже может быть преобразована в полноценную RollTable Foundry через импортёр.
