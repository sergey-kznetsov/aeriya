# Пепельная Степь — дорожные события

## Назначение

Таблица используется мастером во время переходов по Пепельной Степи. Результаты не обязаны быть боевыми: часть событий создаёт выбор, задержку, слух или социальную сцену.

## d12

| d12 | Событие |
|---:|---|
| 1 | Караванный след уходит с известной дороги. |
| 2 | Путники находят старый дорожный знак, частично занесённый пеплом. |
| 3 | Ветер приносит обрывок чужого письма. |
| 4 | У стоянки ждёт человек, который называет неправильное имя. |
| 5 | Проводник просит сменить маршрут без долгих объяснений. |
| 6 | Вдалеке виден пепельный маяк, которого нет на карте. |
| 7 | На дороге стоят пустые телеги без следов владельцев. |
| 8 | Местные предлагают чай и проверяют реакцию гостей. |
| 9 | Один из животных не хочет идти дальше. |
| 10 | Караванные метки указывают в две стороны одновременно. |
| 11 | На стоянке найден свежий костёр, но людей рядом нет. |
| 12 | Дорога приводит к месту, которое должно быть далеко отсюда. |

## Использование в Foundry
