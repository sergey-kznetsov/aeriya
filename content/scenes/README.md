# Сцены и локации

Раздел для сцен, боевых площадок, важных помещений, дорог, стоянок, руин, храмов, рынков, входов в город и иных мест, которые мастер может использовать в игре.

Сцены в `content/` описывают игровую логику и атмосферу. Карты и изображения добавляются вручную в готовом мире Foundry и не хранятся в репозитории по умолчанию.

Рекомендуемые подпапки:

```text
by-region/
by-city/
roads/
combat-arenas/
ritual-sites/
ruins/
```

Карточка сцены должна содержать:

```text
Название
Регион
Тип сцены
Описание для игроков
Скрытая информация мастера
Кто здесь может быть
Какие фракции связаны
Какие монстры возможны
Какие handouts можно выдать
Какие проверки уместны
Как сцена меняется после действий партии
```
