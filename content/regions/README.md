# Регионы

Раздел для Осколков, регионов, дорог, природных зон и опасных территорий Аэрии.

Материалы региона должны помогать мастеру быстро понять атмосферу места, правила выживания, ключевые угрозы, связанные города, фракции, духов, монстров и дорожные события.

Рекомендуемые подпапки:

```text
sredinnye-zemli/
tenevoy-oskolok/
palyashchiy-oskolok/
mezhoskolkovye-dorogi/
```

Карточка региона должна содержать:

```text
Название
Осколок
Тон региона
Климат
Правила выживания
Ключевые города
Фракции
Духи
Монстры
Опасности
Дорожные события
Слухи
Handouts
Сцены
Что игроки видят сразу
Что знает местный житель
Что скрыто от игроков
```
