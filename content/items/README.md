# Предметы и артефакты

Раздел для обычных предметов, редкостей, реликвий, артефактов, расходников, трофеев и региональных диковин Аэрии.

Предметы должны быть связаны с регионом, фракцией, историей, ритуалом или монстром. В модуль не добавляются случайные внешние предметы, если они не встроены в лор Аэрии.

Рекомендуемые подпапки:

```text
common/
rare/
artifacts/
ritual-components/
monster-trophies/
regional-items/
```

Карточка предмета должна содержать:

```text
Название
Тип
Редкость
Регион
Краткое описание
Кто использует
Откуда берётся
Игровой эффект
Цена или риск
Связанные фракции
Связанные ритуалы
Связанные монстры
```
