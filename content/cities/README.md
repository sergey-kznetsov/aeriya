# Города и поселения

Раздел для городов, поселений, стоянок, форпостов и ключевых обитаемых точек Аэрии.

Материалы должны быть разделены по Осколкам и регионам. В карточке города нужно хранить не только описание, но и игровые связи: фракции, NPC, опасности, тайны, сцены и handouts.

Рекомендуемые подпапки:

```text
sredinnye-zemli/
tenevoy-oskolok/
palyashchiy-oskolok/
```

Карточка города должна отвечать на вопросы мастера: кто здесь правит, что видят игроки, какие фракции действуют внутри, кого можно встретить, какие опасности есть рядом и какие материалы можно выдать игрокам.
