# Красная Соль

## Кратко

Красная Соль — поселение Палящего Осколка, связанное с соляной дорогой, запасами, обменом и проводниками.

## Власть

Поселением управляют старшие домов, смотрители запасов и мастера дороги.

## Роль в игре

Место подходит для покупки припасов, проверки маршрута, найма проводника и разговора с местными жителями.

## Зацепки

- Один из знаков дороги изменился.
- Караван ждёт решения старших.
- Местный мастер просит проверить старую площадку.

## Использование в Foundry

Journal Entry для поселения. Позже может быть связан с таблицами дорог и НИПами Палящего Осколка.
