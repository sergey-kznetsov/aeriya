# Города и поселения Палящего Осколка

Здесь будут лежать города, оазисы, заставы, подземные поселения и опасные обитаемые точки Палящего Осколка.

Внутреннее деление:

```text
burnt-wastes/     — Выжженные пустоши
salt-deserts/     — Соляные пустыни
fire-canyons/     — Огненные каньоны
underground/      — подземные поселения и Хранители Пещер
```

Каждый город оформляется по шаблону:

```text
content/_templates/city.md
```
