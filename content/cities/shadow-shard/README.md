# Города и поселения Теневого Осколка

Здесь будут лежать города, стойбища, лагеря, форпосты и скрытые локации Теневого Осколка.

Внутреннее деление:

```text
ash-steppe/              — Пепельная Степь
black-whisper-forest/    — Лес Чёрного Шёпота
icebound-reaches/        — Ледяной / Морозный Предел
```

Каждый город оформляется по шаблону:

```text
content/_templates/city.md
```
