# Города и поселения Срединных Земель

Здесь будут лежать карточки городов, поселений, форпостов и скрытых локаций Срединных Земель.

Примеры материалов:

```text
graven-hollow.md
kladovaya-perekrestkov.md
mornsted.md
farn-krot.md
ozernik.md
plaven.md
skrezhetnik.md
byushchiy-kolokol.md
seryy-priyut.md
```

Каждый город оформляется по шаблону:

```text
content/_templates/city.md
```
