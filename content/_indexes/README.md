# Индексы контента

Папка `_indexes` предназначена для навигационных файлов, которые помогают быстро находить материалы во время игры и при сборке модуля.

Индексы не заменяют сами карточки городов, фракций, НИПов, монстров и ритуалов. Они нужны как оглавление.

Планируемые индексы:

```text
navigation-map.md     — общая карта структуры контента
city-index.md         — список городов и поселений
faction-index.md      — список фракций
npc-index.md          — список НИПов
monster-index.md      — список существ и монстров
ritual-index.md       — список ритуалов
spirit-index.md       — список духов
handout-index.md      — список внутриигровых текстов
```
