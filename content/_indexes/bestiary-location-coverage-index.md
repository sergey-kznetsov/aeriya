# Матрица бестиария по локациям Аэрии

## Назначение

Этот индекс проверяет бестиарий не общим списком, а по каждой игровой локации: у каждой локации должен быть хотя бы один монстр / угроза и понятный статус чарника.

Статусы:

- `actor-ready` — статблок уже добавлен в `content/actor-statblocks`;
- `source-ready` — статблок есть в исходном бестиарии и должен быть перенесён без перепридумывания;
- `new-actor-needed` — существо новое для модуля, нужен новый actor-ready статблок;
- `external-closed-node` — локация закрыта по внутренней части, используется только внешний периметр.

## Вывод аудита

Локационное покрытие угрозами есть по всем крупным локациям, но не все угрозы имеют actor-ready статблоки в модуле.

Критичные пробелы:

1. Новые существа Срединных Земель — 9 статблоков.
2. Модульные существа Теневого Осколка, которые не совпадают напрямую с исходниками.
3. Модульные существа Палящего Осколка, которые уже используются в локациях, но ещё не перенесены в actor-ready формат.

Следующее действие: закрыть `Bestiary new batch 01` для новых существ Срединных Земель, затем `Bestiary source/new batch` для Теневого и Палящего Осколков.
