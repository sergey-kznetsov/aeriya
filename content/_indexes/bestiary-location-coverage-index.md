# Матрица бестиария по локациям Аэрии

## Назначение

Этот индекс проверяет бестиарий не общим списком, а по каждой игровой локации: у каждой локации должен быть хотя бы один монстр / угроза и понятный статус чарника.

Статусы:

- `actor-ready` — статблок уже добавлен в `content/actor-statblocks`;
- `source-ready` — статблок есть в исходном бестиарии и должен быть перенесён без перепридумывания;
- `module-creature-pending` — существо уже используется в модуле, но его actor-ready статблок ещё не добавлен;
- `external-closed-node` — локация закрыта по внутренней части, используется только внешний периметр.
