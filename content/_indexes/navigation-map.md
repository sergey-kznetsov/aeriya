# Навигационная карта контента

Этот файл фиксирует целевую структуру контента Aeria Core. Все материалы пересобираются с нуля из документации Аэрии и не копируются напрямую из старого мира `shadow-lands`.

## Основные разделы

```text
content/
├── regions/
├── cities/
├── factions/
├── npcs/
├── monsters/
├── items/
├── rituals/
├── spirits/
├── handouts/
├── rolltables/
├── scenes/
├── _templates/
└── _indexes/
```

## Деление по Осколкам

Для крупных разделов используется одинаковое деление:

```text
middle-lands/      — Срединные Земли
shadow-shard/      — Теневой Осколок
scorching-shard/   — Палящий Осколок
all-shards/        — материалы, действующие во всех Осколках
outside/           — Извне, Завязь, межосколочные угрозы
```

## Теневой Осколок

```text
shadow-shard/
├── ash-steppe/              — Пепельная Степь
├── black-whisper-forest/    — Лес Чёрного Шёпота
└── icebound-reaches/        — Ледяной / Морозный Предел
```

## Палящий Осколок

```text
scorching-shard/
├── burnt-wastes/            — Выжженные пустоши
├── salt-deserts/            — Соляные пустыни
├── fire-canyons/            — Огненные каньоны
└── underground/             — подземные города, пещеры и Хранители Пещер
```

## Срединные Земли

```text
middle-lands/
├── trade-cities/            — торговые города и крупные узлы
├── wetlands/                — болота, водные города, Плавень и Озерник
├── mines-and-mountains/     — шахтёрские города и горные локации
├── roads/                   — дороги, караванные стоянки, переходы
└── hidden-locations/        — скрытые и особые локации
```

## Правило поиска во время игры

Материал должен лежать там, где мастер будет искать его первым.

Пример: если НИП живёт в Гравен Холлоу и связан с Храмом Последней Торговки, его основной файл лежит в `content/npcs/middle-lands/`, а ссылки на него добавляются в карточку города и карточку фракции.

Пример: если монстр встречается во всех Осколках, он лежит в `content/monsters/all-shards/`. Если это существо Леса Чёрного Шёпота, оно лежит в `content/monsters/shadow-shard/black-whisper-forest/`.

## Что не попадает в структуру

```text
старые внешние compendium packs;
музыка;
пользователи;
история чата;
fog;
настройки старого мира;
готовые токены и портреты;
карты без ручной проверки;
любой контент, не относящийся к Аэрии.
```
