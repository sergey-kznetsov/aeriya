# Общие ритуалы

Здесь будут лежать ритуалы, которые может попытаться провести любой персонаж при наличии инструкции, слов, компонентов, времени, места и понимания риска.

Примеры категорий:

```text
ritual-of-safe-camp.md
ritual-of-road-sign.md
ritual-of-spirit-request.md
ritual-of-burial.md
```

Шаблон:

```text
content/_templates/ritual.md
```
