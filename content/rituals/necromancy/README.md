# Некромантические ритуалы

Здесь будут лежать ритуалы, связанные со смертью, душами, телами, памятью мёртвых, погребальными обрядами, обменом души на душу и возможным нарушением воли Омулу.

Некромантический ритуал всегда должен иметь цену, риск или долг.

Шаблон:

```text
content/_templates/ritual.md
```

При описании обязательно заполнять разделы:

```text
Цена
Откат
Связанные духи
Некромантическая версия
Последствия нарушения обряда
```
