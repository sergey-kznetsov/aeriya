# Ритуалы

Раздел для ритуалов Аэрии, некромантических практик, требований, компонентов, слов, мест силы и откатов.

Ритуалы может проводить любой персонаж, если у него есть инструкция, нужные слова, компоненты, время, подходящее место и понимание риска.

Рекомендуемые подпапки:

```text
common-rituals/
spirit-rituals/
necromancy/
road-rituals/
protection/
backlash-tables/
```

Карточка ритуала должна содержать:

```text
Название
Тип
Кто может провести
Требования
Компоненты
Нужные слова
Время
Место
Сложность
Успех
Частичный успех
Провал
Откат
Связанный дух
Связанная фракция
Связанные handouts
```
