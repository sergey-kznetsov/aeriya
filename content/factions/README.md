# Фракции

Раздел для фракций, организаций, культов, караванов, орденов, гильдий и иных сил Аэрии.

Фракции хранятся отдельно от городов, чтобы мастер мог быстро открыть карточку силы и увидеть её цели, методы, лидеров, союзников, врагов и отношение к партии.

Рекомендуемые подпапки:

```text
sredinnye-zemli/
tenevoy-oskolok/
palyashchiy-oskolok/
mezhoskolkovye/
```

Каждая карточка фракции должна содержать зоны влияния, связанные города, важных NPC, союзников, противников, способы поднять или испортить репутацию и сюжетные крючки.
