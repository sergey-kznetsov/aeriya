# Фракции Срединных Земель

Здесь будут лежать фракции, торговые союзы, культы, гильдии, отряды и преступные сети Срединных Земель.

Примеры материалов:

```text
black-caravan.md
fifth-tooth-guild.md
oath-blades.md
last-trader-temple.md
hunters.md
roof-worms.md
```

Каждая фракция оформляется по шаблону:

```text
content/_templates/faction.md
```
