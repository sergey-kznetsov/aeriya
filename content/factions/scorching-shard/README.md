# Фракции Палящего Осколка

Здесь будут лежать фракции, культы, водные союзы, кочевники, техномаги, стражи, пираты пустыни и подземные группы Палящего Осколка.

Внутреннее деление:

```text
burnt-wastes/     — города, оазисы, пустошные фракции
salt-deserts/     — Солегрызы, исследователи, Соляные Стражи
fire-canyons/     — добытчики, стеклодувы, огненные культы
underground/      — Хранители Пещер и подземные силы
```

Каждая фракция оформляется по шаблону:

```text
content/_templates/faction.md
```
