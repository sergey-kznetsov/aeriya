# Фракции Теневого Осколка

Здесь будут лежать фракции Пепельной Степи, Леса Чёрного Шёпота и Ледяного / Морозного Предела.

Внутреннее деление:

```text
ash-steppe/              — степные города, кочевники, караваны, шаманские группы
black-whisper-forest/    — Стражи Тропы, лесные культы, отшельники
icebound-reaches/        — ледяные кланы, стража, культы и городские силы
```

Каждая фракция оформляется по шаблону:

```text
content/_templates/faction.md
```
