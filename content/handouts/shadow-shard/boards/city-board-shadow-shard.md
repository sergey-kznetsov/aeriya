# Доски поручений Теневого Осколка

## Назначение

Сводная доска поручений по трём крупным регионам Теневого Осколка: Пепельная Степь, Лес Чёрного Шёпота и Вечные Льды.

## Пепельная Степь

| Место | Доска поручений |
|---|---|
| Три Тени Перекрёстка | Разобраться, почему караван выбрал не ту тень у развилки; проверить запись стоянки; найти проводника, который ушёл без чая. |
| Чёрная Кузница | Доставить уголь без спора с мастерами; найти пропавший инструмент; узнать, кто заказал оружие без имени заказчика в записи. |
| Аалмар | Проверить торговый спор; найти свидетеля за второй чашей; сопроводить посредника до совета. |
| Ташвейн | Найти пропавший знак у дороги; проверить старый чайный дом; договориться с водными смотрителями. |
| Борос | Разобрать конфликт каравана и стражи; проверить склад; найти человека, который переписал путь. |
| Бату | Найти пастуха после пепельного ветра; проверить новый след у колодца; вернуть знак семьи. |
| Сайрам | Узнать, кто подал пятую чашу чужаку; найти пропавшую чашу; примирить два дома. |
| Ак-Кул | Проверить белый берег; найти место ночной стоянки; выяснить, почему вода стала горькой. |
| Кум-Алам | Провести караван через серые холмы; найти старую метку; вернуть ребёнка с опасного склона. |
| Хурнаир | Проверить маяк; найти снятый фонарь; предупредить караван до выхода. |
| Ип | Выяснить, кто оставляет знаки на маленькой стоянке; найти потерянного путника; проверить чайную запись. |
| Пепельное Сердце | Найти след старого огня; проверить спор Совета Пяти Клинков; поговорить с духом дороги через проводника. |
| Гар-Шахр | Проверить ворота старого поселения; найти свидетеля у кузницы; понять, почему караваны обходят город. |
| Окрам | Статус: требует сверки со старым миром. До сверки использовать только как слух о закрытом месте. |

## Лес Чёрного Шёпота

| Место | Доска поручений |
|---|---|
| Корень Тёмного Мха | Найти сбитую тропу; проверить мховую метку; договориться с Домом Мха. |
| Сердечная Поляна | Узнать, кто нарушил молчание; вернуть знак проводника; проверить шёпот у старого дерева. |
| Старая Лесная Застава | Найти запись заставника; проверить след без тела; вывести путников с неверной тропы. |
| Серый Приют | Доставить просьбу без громких слов; найти человека, которого лес не выпускает; проверить серую часовню. |

## Вечные Льды

| Место | Доска поручений |
|---|---|
| Ледяной Клык | Проверить треснувший знак; найти пропавшего проводника; договориться с Ледяными Скульпторами. |
| Станция Последний Огонь | Проверить гаснущий очаг; сверить записи дежурства; найти того, кто подходил к огню ночью. |
| Перевал Синего Ветра | Вернуть снятую метку; проверить тонкий лёд; вывести караван на безопасный путь. |
| Лагерь Белых Камней | Найти следы у камней; проверить исчезнувший запас; переждать северный ветер без ссоры у очага. |

## Быстрое использование

1. Выбрать регион.
2. Взять место и одно поручение.
3. Привязать местную фракцию.
4. Добавить региональное существо.
5. Если место ещё не полностью оформлено, использовать поручение как временный игровой мост.
