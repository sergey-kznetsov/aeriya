# Доски поручений Срединных Земель

## Назначение

Сводная доска поручений по городам и поселениям Срединных Земель. Нужна, чтобы у каждого города была стартовая игровая зацепка до появления отдельной цепочки квестов.

## Поручения по городам

| Город / поселение | Доска поручений |
|---|---|
| Гравен Холлоу | Найти человека, который оставляет пустые таблички на дверях; проверить старый подвал под архивом; выяснить, кто подменил список умерших. |
| Морнстэд | Вернуть пропавшую дорожную книгу; сопроводить писца до рынка; проверить ночной спор у складов. |
| Кладовая Перекрёстков | Найти груз без владельца; проверить караванную печать; провести тихую сделку без фракционного конфликта. |
| Фарн’Крот | Спуститься к нижнему ярусу; найти пропавшего шахтёра; проверить, почему старые балки начали петь. |
| Озерник | Узнать, кто меняет уровень шлюзов ночью; найти лодку без гребца; договориться с рыбаками о проходе. |
| Плавень | Отметить безопасные мостки; найти ребёнка у старой заводи; проверить огни над трясиной. |

| Усадьба Леденёвых | Доставить письмо без печати; проверить слугу, который знает слишком много; найти вещь в закрытом саду. |

## Быстрое использование

1. Выбрать город.
2. Взять одно поручение из строки.
3. Привязать НИПа из общего индекса Срединных Земель.
4. Добавить местную фракцию или угрозу.
5. После сессии решить, превращать ли поручение в отдельный квест.
