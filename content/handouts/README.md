# Handouts

Раздел для внутриигровых текстов, которые можно выдавать игрокам: книг, баек, преданий, легенд, пророчеств, записок, писем, фрагментов карт и дорожных предупреждений.

Handouts не должны содержать мастерских технических комментариев, если эти комментарии не скрыты отдельно.

Рекомендуемые подпапки:

```text
road-tales/
legends/
traditions/
prophecies/
fire-singer/
notes/
letters/
fragments/
```

Карточка handout должна содержать:

```text
Название
Тип текста
Регион
Кто мог написать
Где можно найти
Текст для игроков
Скрытый смысл для мастера
Связанные города
Связанные фракции
Связанные NPC
Связанные ритуалы
```
