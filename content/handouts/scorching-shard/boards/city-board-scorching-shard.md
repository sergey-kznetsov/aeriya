# Доски поручений Палящего Осколка

## Назначение

Сводная доска поручений по ключевым городам и локациям Палящего Осколка. Используется как быстрый слой заданий до превращения каждого места в отдельный квестовый узел.

## Поручения по городам и локациям

| Место | Доска поручений |
|---|---|
| Три Тени | Найти караван, который видели сразу в трёх местах; проверить спор трёх проводников; выяснить, кто поменял метки на развилке. |
| Пылающий Клинок | Доставить воду к кузнецам; найти исчезнувший клинок без рукояти; остановить драку у горячих мастерских. |
| Стеклянный Городок | Вернуть стеклянный образец; найти отражение чужого сигнала; договориться с Мастерскими Линз. |
| Последний Вздох | Проверить последнюю стоянку каравана; найти запись о выходе; переждать сухой ветер в закрытом доме. |
| Пещерный Порог | Сопроводить проводника к нижнему входу; найти потерянную верёвку; проверить звук из старой трещины. |
| Красная Соль | Узнать, кто оставил соляную метку; проверить спор Соляного Круга; найти пропавший мешок лечебной соли. |
| Небесный Якорь | Проверить неверный сигнал; найти человека, поднявшегося к знаку ночью; договориться со смотрителем. |
| Лагерь Шёпот Ветра | Установить, чей голос слышали ночью; вернуть ткань с чужим знаком; провести переговоры между стоянками. |
| Тихая Корка | Найти скрытую трещину под настилом; проверить старую корку соли; вывести ребёнка с закрытой площадки. |
| Форт Раскалённой Соли | Проверить печать каравана; найти сбежавшего свидетеля; решить спор у ворот без драки. |
| Дворец Золотого Каравана | Сопроводить купца на закрытую встречу; проверить цену груза; узнать, кто подменил список даров. |
| Кровавый Родник | Навести порядок в очереди за водой; найти чужую метку у источника; выяснить, кто нарушил запись воды. |
| Руины Соляной Мельницы | Проверить старый механизм; найти карту маршрута; пережить ночной шорох в руинах. |
| Белый Колодец | Найти пропавшее ведро с меткой; проверить глубину после ветра; договориться с хранителем прохода. |
| Каменный Навес | Найти свидетеля под навесом; выяснить, кто оставил тайник; защитить путников от дневной жары. |

## Быстрое использование

1. Выбрать место.
2. Взять одно поручение.
3. Привязать НИПа из Палящего Осколка.
4. Добавить фракцию: Стража Пламени, Соляной Круг, Песчаные Братья, Свободные Капли или Мастерские Линз.
5. При необходимости добавить существо из бестиария Палящего Осколка.

## Использование в Foundry

GM handout и временная доска заданий для городов и локаций Палящего Осколка.
