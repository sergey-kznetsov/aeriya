# Стандарт статблоков Aeria Core

## Назначение

Этот стандарт нужен, чтобы НИПы и существа могли быть быстро перенесены в Foundry VTT как Actor.

## Формат для НИПов

Каждый НИП получает блок:

- роль;
- рекомендуемый класс / архетип;
- уровень;
- CR;
- КД;
- хиты;
- скорость;
- характеристики;
- спасброски;
- навыки;
- чувства;
- языки;
- снаряжение;
- действия;
- бонусные действия / реакции;
- особенности;
- тактика.

## Уровни НИПов

Основная планка для значимых НИПов: 10-15 уровень.

| Тип НИПа | Рекомендованный уровень |
|---|---:|
| локальный специалист / проводник | 10-11 |
| городской силовой НИП / старший мастер | 11-13 |
| глава узла / опасный переговорщик | 13-15 |
| сюжетный лидер / региональный агент | 15+ только по необходимости |

## Формат для существ

Существо получает:

- размер и тип;
- CR;
- КД;
- хиты;
- скорость;
- характеристики;
- спасброски;
- навыки;
- сопротивления / иммунитеты;
- чувства;
- языки;
- действия;
- реакции;
- особенности;
- тактика;
- зацепка для сцены.

## Баланс

Статблоки являются author-ready / actor-ready заготовками. Их можно импортировать как Journal Entry и затем вручную перенести в Actor dnd5e.

Для игровых сцен используются не официальные существа D&D, а кастомные существа Аэрии.
