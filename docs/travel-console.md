# Консоль путешествий Аэрии

Консоль путешествий — мастерский инструмент Foundry VTT для автоматизации переходов между локациями. Она не заменяет решение мастера и игроков, а быстро собирает каркас дороги: роли партии, региональное давление, дорожные кости, узлы перехода, осложнение припасов, случайную встречу и последствия прибытия.

## Как открыть

После активации модуля в мире Foundry создаётся макрос:

```text
Аэрия: Консоль путешествия
```

Также кнопка консоли добавляется в панель инструментов сцены для мастера.

Команда для ручного вызова:

```js
game.aeriya.travel.openConsole();
```

## Что происходит при открытии консоли

Если включена настройка `Аэрия: автоматически распределять дорожные роли`, модуль сразу распределяет дорожные роли и публикует результат в общий чат. Игроки видят, кто за что отвечает в пути.

Порядок выбора персонажей:

```text
1. выбранные токены на сцене;
2. персонажи активных игроков;
3. все player-owned персонажи мира.
```

Роли записываются в:

```text
flags.aeriya.travel.currentRoles
flags.aeriya.travel.lastRoleAssignmentAt
```

Последнее распределение хранится в world setting:

```text
game.settings.get("aeriya", "lastTravelRoles")
```

## Дорожные роли

В текущем MVP используются шесть ролей:

```text
Проводник маршрута — путь, местность, обходы, сокращения.
Дозорный — угрозы, ночной дозор, засады, хвост, странные знаки.
Снабженец — припасы, вода, фляги, лекарства, корм, расходники.
Переговорщик — проводники, караваны, заставы, местные, попутчики.
Хранитель груза — телега, сумки, опасный груз, крепления, животные, пропажи.
Летописец пути — карта, слухи, приметы, долги, договоры, дорожная память.
```

Распределение идёт по подходящим навыкам и характеристикам D&D 5e. Если персонажей меньше, чем ролей, часть персонажей получает дополнительные роли.

## Что генерирует консоль

Основная кнопка `Сгенерировать переход` создаёт:

```text
маршрут из точки А в точку Б;
регион и его давление;
длительность пути;
уровень риска;
дорожные кости по дням пути;
состояния ресурсов: припасы, снаряжение, время, доверие и региональный ресурс;
3–5 узлов перехода;
отдельное осложнение припасов;
случайную встречу;
мастерскую заметку о том, как вести переход.
```

Сгенерированный переход сохраняется в:

```text
game.settings.get("aeriya", "lastTravelJourney")
```

## Регионы

В генератор заложены региональные наборы для:

```text
Срединные Земли;
Пепельная Степь;
Лес Чёрного Шёпота;
Ледяной Предел;
Выжженные пустоши;
Соляные пустыни;
Межосколочные дороги Завязи.
```

Каждый регион имеет собственное давление, набор угроз, фракции, сцены дороги, привала, узкого места, прибытия и порчи припасов.

## Чат и видимость

Дорожные роли публикуются в общий чат, чтобы игроки видели свои обязанности.

Сгенерированные сцены, осложнения, случайные встречи и мастерские последствия отправляются в приватный чат мастера через whisper GM, если мастер нажимает соответствующие кнопки. Это сделано, чтобы не раскрывать игрокам скрытые сцены заранее.

## Journal Entry

Кнопка `Создать журнал перехода` создаёт Journal Entry с последним сгенерированным переходом. Это удобно, если мастер хочет сохранить маршрут как заготовку сцены или повторно использовать его позже.

## Настройки

```text
Аэрия: создать макрос путешествия
Аэрия: автоматически распределять дорожные роли
```

Обе настройки включены по умолчанию.

## Публичный API

```js
// открыть консоль
game.aeriya.travel.openConsole();

// сгенерировать переход без интерфейса
game.aeriya.travel.generateJourney({
  from: "Гравен Холлоу",
  to: "Сычий Брод",
  region: "sredinnye_zemli",
  days: 3,
  danger: 2,
  purpose: "доставить груз",
  cargo: "ящик Чёрного Каравана"
});

// сгенерировать встречу
game.aeriya.travel.generateEncounter("pepelnaya_steppe");

// сгенерировать порчу припасов
game.aeriya.travel.generateSupplyIssue("black_forest");

// распределить роли вручную
game.aeriya.travelRoles.assignAndPost();
```
