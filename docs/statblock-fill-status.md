# Статус заполнения чарников Aeria Core

## Текущий вывод

Бестиарий не нужно создавать с нуля: у исходных бестиариев уже есть D&D 5e статблоки. Для монстров задача — перенос и сопоставление. Для НИПов задача — создание actor-ready статблоков, если готового чарника нет.

Важное уточнение: ранее закрытым было минимальное локационное покрытие — то есть у каждой крупной локации есть хотя бы одна рабочая actor-ready угроза. Это не означает, что полный бестиарий по каждой локации / региону уже перенесён.

NPC-чарники по трём крупным блокам закрыты: Срединные Земли, Теневой Осколок, Палящий Осколок.
