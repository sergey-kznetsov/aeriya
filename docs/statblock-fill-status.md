## Текущий вывод

Бестиарий не нужно создавать с нуля: у исходных бестиариев уже есть D&D 5e статблоки. Для монстров задача — перенос и сопоставление. Для НИПов задача — создание actor-ready статблоков, если готового чарника нет.
