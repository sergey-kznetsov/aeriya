# Прогресс переноса контента

Модуль Аэрии собирается с нуля как надстройка над D&D 5e.

Старые compendium packs не копируются. Новые документы Аэрии являются основным источником. Прежний мир используется только как точечный справочник для отдельных сущностей.

## Текущий статус

### Срединные Земли

Готов первый справочный слой городов и фракций. Добавлен быстрый игровой слой:

- 8 НИПов;
- 4 сцены;
- 3 таблицы;
- 2 поручения;
- 2 мини-сценария;
- player-facing handout;
- GM handout;
- отдельный игровой индекс Срединных Земель.

Следующее: при необходимости расширить мини-сценарии, но приоритетнее перейти к Foundry packs.

### Теневой Осколок

Готов базовый игровой слой по трём регионам:

- Пепельная Степь;
- Лес Чёрного Шёпота;
- Вечные Льды.

Есть регионы, локации, НИПы, существа, предметы, сцены, таблицы, духи и ритуалы. Чайная традиция Пепельной Степи вынесена в handout.

Следующее: добавить слой поручений и мини-сценариев по регионам, затем закрыть сверку Окрама.

### Палящий Осколок

Готов самый полный игровой слой:

- региональные карточки;
- 15 городов и локаций;
- фракции;
- НИПы;
- предметы;
- сцены;
- существа;
- духи;
- ритуалы;
- таблицы;
- поручения;
- памятки;
- мини-сценарии.

Следующее: не расширять без необходимости, а переводить контент в Foundry packs.

## Технический статус

Уже есть:

- `module.json`;
- JS-слой модуля;
- русская локализация;
- стили;
- валидатор карточек;
- сборщик Journal Entry JSON;
- структура `assets/`;
- asset manifest.

Не хватает до хорошего модуля:

- реальные compendium packs;
- обновление `module.json` с packs;
- release zip;
- тест в чистом мире Foundry;
- первые Actor/Item/RollTable/Scene документы вместо markdown-заготовок.

## Ближайший приоритет

1. Закрыть быстрый слой поручений и мини-сценариев Теневого Осколка.
2. Добавить Foundry pack/export план.
3. Сделать первый `aeriya-journals` pack или максимально близкий к нему экспорт.
4. После стабильного Journal Entry pack переходить к картинкам, картам, портретам и токенам.
