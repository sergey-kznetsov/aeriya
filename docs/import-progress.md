# Прогресс переноса контента

Модуль Аэрии собирается с нуля как надстройка над D&D 5e.

Старые compendium packs не копируются. Новые документы Аэрии являются основным источником. Прежний мир используется только как точечный справочник для отдельных сущностей.

## Порядок

1. Перенос карточек малыми коммитами в `main`.
2. Разделение по типам: города, фракции, НИПы, монстры, духи, предметы, ритуалы, handouts и таблицы.
3. Разделение по Осколкам и регионам.
4. Не добавлять пользовательские данные прежнего мира, историю чата, fog, настройки мира, музыку и внешние пакеты.
5. После пачек обновлять индекс контента.
6. Перед импортом в Foundry проверять карточки валидатором.

## Статус

### Срединные Земли

Первый слой городов и базовых фракций заведён и проиндексирован.

### Теневой Осколок — Пепельная Степь

Заведён первый игровой слой региона:

- региональная справка;
- города и поселения;
- часть фракций;
- НИПы для дороги, чая, стоянок, записей и дозора;
- стартовый бестиарий;
- дорожные таблицы;
- предметы дороги;
- сцены-заготовки для Foundry.

Окрам оставлен на отдельную сверку со старым миром `shadow-lands`. Гар-Шахр и часть фракций Пепельной Степи ожидают альтернативного безопасного переноса через упрощённые карточки.

### Теневой Осколок — Лес Чёрного Шёпота

Первый игровой слой региона расширен:

- региональная справка;
- поселения и локации;
- НИПы для троп, мха, слухов и старой заставы;
- стартовый лесной бестиарий;
- таблицы лесных сигналов и сбившихся троп;
- локальные группы;
- предметы для троп и проводников;
- духи региона;
- первый лесной обряд;
- сцены-заготовки для Foundry.

Индекс таблиц для лесного блока пока требует отдельного безопасного обновления: сами таблицы уже добавлены, но обновление общего файла индекса было заблокировано инструментом.

### Следующий этап

Закрыть оставшиеся элементы Леса Чёрного Шёпота: проблемные карточки, дополнительные предметы и ритуалы. Затем перейти к Вечным Льдам.
