# Установка Aeria Core в новый мир Foundry

Основной сценарий использования модуля: создать новый чистый мир Foundry и подключить к нему Aeria Core.

## Шаги

```text
1. Открыть Foundry VTT.
2. Создать новый мир.
3. В качестве системы выбрать D&D 5e.
4. Установить модуль Aeria Core.
5. Активировать модуль в настройках мира.
6. Проверить, что доступны правила, справочники, будущие compendium packs и инструменты модуля.
7. Вручную добавить карты, сцены, изображения, портреты и токены.
```

## Что модуль должен принести в мир

```text
правила Аэрии;
боевой слой;
реакции;
точные попадания;
ритуалы;
духов;
фракции;
города;
NPC;
монстров;
предметы;
handouts;
таблицы;
инструменты мастера.
```

## Что модуль не должен приносить

```text
старый мир shadow-lands;
старые compendium packs;
пользовательские данные;
историю чата;
fog of war;
старые настройки кампании;
аудио;
карты;
портреты;
токены;
случайный внешний контент.
```

## Почему так

Модуль должен быть чистым и переиспользуемым. Его задача — дать новому миру Foundry структуру и игровые материалы Аэрии, а не переносить старую кампанию.

Визуальные материалы добавляются отдельно, потому что они могут меняться от кампании к кампании и не должны смешиваться с логикой модуля.
