# Аудит готовности модуля Aeria Core

Дата фиксации: 2026-06-25.

## Краткий вывод

Модуль уже имеет рабочую основу: `module.json`, JS-слой, стили, русскую локализацию, структуру контента, валидатор, экспорт markdown-карточек в JSON и большой набор игровых материалов.

Главная проблема не в лоре, а в Foundry-готовности. Контент в основном хранится как markdown-карточки. Для быстрого доведения до рабочего состояния нужно сначала сделать нормальную сборку Journal Entry pack и навигацию, а уже потом превращать часть карточек в Actor, Item, RollTable и Scene.

## Состояние по крупным блокам

| Блок | Состояние | Что есть | Главный пробел |
|---|---|---|---|
| Срединные Земли | средний слой | много городов, базовые фракции, часть духов | мало НИПов, сцен, таблиц, поручений и игровых связок |
| Теневой Осколок | хороший базовый слой | регионы, города, фракции, НИПы, монстры, предметы, сцены, духи, ритуалы | нужен слой handouts, мини-сценариев, сверка Окрама и актуализация старых хвостов |
| Палящий Осколок | самый готовый игровой слой | регионы, 15 локаций, фракции, НИПы, предметы, сцены, таблицы, поручения, мини-сценарии | нужно не расширять бесконечно, а переводить в Foundry packs |
| Технический Foundry-слой | базовый | module.json, локализация, JS API, валидатор, journal export | packs пустые, нет релизной сборки, markdown ещё не превращён в compendium |

## Срединные Земли

### Уже хорошо

- Есть первый слой городов и поселений.
- Есть фракции Срединных Земель.
- Есть базовые духи и часть связей с общим лором.
- Регион хорошо подходит как стартовая зона и центр навигации между Осколками.

### Чего не хватает для хорошего модуля

- НИПы для ключевых городов: трактирщики, проводники, главы фракций, торговцы, стража, информаторы.
- Мини-сценарии на 2-4 сцены.
- Таблицы городских слухов, дорожных встреч, осложнений в поселениях.
- Player-facing handouts: что знают жители Срединных Земель, обычаи дорог, памятка по фракциям.
- Сцены-заготовки: рынок, городские ворота, трактир, старый мост, храм, склад, лесная тропа.

### Быстрая добивка

Минимум для рабочего слоя: 8 НИПов, 4 сцены, 3 таблицы, 4 поручения, 2 мини-сценария.

## Теневой Осколок

### Уже хорошо

- Пепельная Степь имеет регион, города, НИПов, предметы, сцены, таблицы и стартовый бестиарий.
- Лес Чёрного Шёпота имеет регион, поселения, НИПов, лесных существ, таблицы, сцены, духов и ритуалы.
- Вечные Льды имеют регион, станции, НИПов, существ, таблицы, сцены, предметы, фракции, духов и ритуалы.
- Чайная традиция Пепельной Степи вынесена в player-facing handout.

### Чего не хватает для хорошего модуля

- Общий слой мини-сценариев по Теневому Осколку.
- Поручения по Пепельной Степи, Лесу Чёрного Шёпота и Вечным Льдам.
- Сверка Окрама со старым миром `shadow-lands`.
- Актуализация статуса старых заблокированных карточек: часть уже внесена через Git tree, документация об этом устарела.
- Несколько боевых Actor-заготовок для ключевых существ.

### Быстрая добивка

Минимум для рабочего слоя: 6 поручений, 3 мини-сценария, 3 player-facing handouts, 3 мастерские памятки, сверка Окрама отдельным пунктом.

## Палящий Осколок

### Уже хорошо

- Есть региональные карточки: Палящий Осколок, Выжженные Пустоши, Соляные дороги, Пещерные маршруты.
- Есть 15 городов и локаций.
- Есть фракции, НИПы, предметы, сцены, таблицы, духи, ритуалы.
- Есть памятки, 5 поручений и 4 мини-сценария.
- Есть таблицы слухов, находок, осложнений, реакций фракций и поворотов мини-сценария.

### Чего не хватает для хорошего модуля

- Не нужно дальше раздувать этот блок до бесконечности.
- Нужно перевести его в реальные Foundry-документы.
- Нужны первые Actor/Item/RollTable/Scene тесты.
- Нужны поля для будущих картинок, портретов, токенов и сцен.

### Быстрая добивка

Минимум для рабочего слоя: подготовить asset manifest и Foundry pack экспорт.

## Технический слой Foundry

### Уже хорошо

- Модуль подключается как add-on поверх D&D 5e.
- Есть `flags.aeriya`.
- Есть настройки реакций, точных попаданий и ритуалов.
- Есть валидатор markdown-карточек.
- Есть сборщик journal JSON.

### Чего не хватает

- `packs` в module.json пока пустой.
- Нет настоящих compendium packs.
- Нет release zip.
- Нет manifest/release flow.
- Нет asset manifest для картинок, портретов, токенов и карт.
- Нет тестового списка: что проверять в чистом мире Foundry.

## Быстрый путь до хорошего модуля

1. Обновить статус-документацию и закрыть устаревшие пометки.
2. Сделать asset manifest: где нужны картинки, карты, портреты и токены.
3. Добавить быстрый слой недостающего контента для Срединных Земель и Теневого Осколка.
4. Сделать первый pack-план и добавить packs в module.json.
5. Подготовить экспорт всех markdown-карточек как Journal Entry pack.
6. Отдельно подготовить первые RollTable-документы.
7. Отдельно подготовить первые Actor-заготовки существ.
8. Проверить модуль в чистом мире Foundry.
9. После этого можно переходить к картинкам, картам и токенам.

## Что делать прямо сейчас

Не расширять Палящий Осколок. Следующий рабочий этап: подтянуть Срединные Земли и Теневой Осколок до минимального игрового слоя, затем заняться Foundry pack/export.
