# Aeria module patch

Пакет для ручной вставки в репозиторий `sergey-kznetsov/aeriya`.

## Как вставить

1. Распаковать архив.
2. Открыть папку `repo-root`.
3. Скопировать содержимое `repo-root` в корень репозитория `aeriya` с заменой файлов.
4. Выполнить:
   ```bash
   npm run content:check
   npm run journals:build
   ```
5. Проверить diff и сделать обычный commit.

## Что внутри

- Карточки, которые блокировались GitHub-коннектором.
- Технические правки валидатора и экспортёра Journal Entry.
- Шаблон региона.
- Индексы и статусные файлы для ручной сверки.
