## Что ещё нужно добавить до локального Foundry-теста

### Приоритет 1. НИПы по городам Срединных Земель

Сейчас есть общий слой НИПов, но не у каждого города есть свой локальный персонаж. Нужно добавить партиями:

- северные/сухие города;
- болотные и речные поселения;
- промышленные и складские города;
- колокольные/домовые/фракционные центры.

Минимум: 1 локальный НИП на каждый город Срединных Земель.

### Приоритет 2. Городские сцены

Сейчас есть 4 универсальные сцены Срединных Земель. Нужно добавить сцены по группам:

- рынок и склады;
- ворота и трактир;
- мосты и переправы;
- болота и сухие берега;
- промышленные дворы;
- колокольни и усадьбы.

### Приоритет 3. Квесты по городским группам

Доски поручений дают временное покрытие, но отдельные quest-card нужны для важных узлов:

- Морнстэд / Кладовая Перекрёстков;
- Фарн’Крот / Скрежетник / Варница Чёрного Дыма;
- Плавень / Рубеж Трясин / Сычий Брод;
- Бьющий Колокол / Усадьба Леденёвых;
- Берег Сухих Глаз / Пустотень / Острог Прахобоя.

### Приоритет 4. Shadow Shard точечные хвосты

- Окрам остаётся на сверке со старым миром.
- Для части мест нужны отдельные city-card / scene-card / quest-card, а не только строка в доске поручений.

### Приоритет 5. Foundry-конверсия

После выравнивания покрытия:

- монстры → Actor;
- предметы → Item;
- таблицы → RollTable;
- сцены → Scene;
- handouts и книги → Journal Entry.

## Следующее действие

До локального тестирования в Foundry продолжать наполнение партиями:

1. Локальные НИПы Срединных Земель.
2. Локальные квесты Срединных Земель.
3. Уточнение Теневого Осколка по Окраму и ключевым местам.
4. Только после этого запускать `npm run foundry:check` и ручной smoke-test.
