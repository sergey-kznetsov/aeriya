# Content Batch 01

Первый чистый пакет контента для Aeria Core.

## Источники

```text
Города и поселения Аэрии.docx
Фракции Аэрии.docx
Духи Аэрии расширенный справочник.docx
```

## Состав пакета

```text
города и поселения: 54
фракции: 37
духи: 29
```

## Что не переносится

```text
старые compendium packs из shadow-lands;
музыка;
пользователи мира;
fog of war;
сообщения чата;
настройки старого мира;
сцены и ассеты;
внешний контент, не связанный с Аэрией.
```

## Правило по Окраму

Окрам переносится как отдельное исключение. Для него новая документация является базой, но финальная версия должна сверяться со старым миром `shadow-lands`, потому что старая интерпретация Окрама считается более точной.

## Индексация

После добавления Markdown-карточек нужно выполнить:

```bash
node scripts/dev/build-content-index.mjs
```

Скрипт обновит:

```text
content/_indexes/content-index.json
```
