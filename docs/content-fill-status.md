# Статус наполнения Aeria Core

## Текущий вывод

Срединные Земли закрыты по базовой игровой матрице. Для каждого из 21 города / поселения есть:

- city card;
- локальный НИП;
- доска поручений;
- квестовая цепочка;
- сцена или specific-scene;
- монстр / угроза.

## Статус по крупным блокам

| Блок | Статус | Что есть | Что осталось |
|---|---|---|---|
| Срединные Земли | content-core-complete | 21 город, 21 локальный НИП, доски, 6 городских квестовых цепочек, 9 монстров, 4 базовые сцены, 19 specific-scenes | только опциональная детализация и будущая Foundry-конверсия |
| Теневой Осколок | mostly-filled | регионы, фракции, монстры, сцены, доска поручений, handouts, мини-сценарии | Окрам, 4-6 city/location quest cards для Леса и Льдов |
| Палящий Осколок | filled-playable | 15 городов/локаций, фракции, НИПы, монстры, поручения, мини-сценарии, доска, сцены, таблицы | опционально 4-6 отдельных city-quest cards по ключевым местам |

## Сколько осталось наполнить до локального Foundry-теста

Минимально обязательное:

| Задача | Количество |
|---|---:|
| Разобрать Окрам / зафиксировать временный статус без старого мира | 1 |
| Добавить city/location quest cards для Леса Чёрного Шёпота | 2-3 |
| Добавить city/location quest cards для Вечных Льдов | 2-3 |
| Обновить индексы после этого | 2-3 |

Итого минимально осталось: примерно 7-10 небольших файлов / правок.

## Следующий шаг наполнения

1. Закрыть Окрам как временный или сверенный узел.
2. Добавить квестовые цепочки Леса Чёрного Шёпота.
3. Добавить квестовые цепочки Вечных Льдов.
